# One Piece Devil Fruits Catalog

Devil Fruits are mystical, mysterious fruits found throughout the world that, when consumed, grant the eater a special super-power, at the cost of losing their ability to swim.

## The Three Main Classes

1. **Paramecia**: Fruits that affect the user's body, environment, or produce substances.
   - Read more in [Paramecia Fruits](paramecia.md).
2. **Zoan**: Fruits that allow the user to transform into an animal or an animal-human hybrid.
   - Read more in [Zoan Fruits](zoan.md).
3. **Logia**: Fruits that allow the user to transform their body composition into a natural element.
   - Read more in [Logia Fruits](logia.md).

For the legendary classification transition, see the mystical [Hito Hito no Mi, Model: Nika](nika.md).
