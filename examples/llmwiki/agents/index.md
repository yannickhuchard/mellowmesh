# Agentic Architecture and Orchestration Patterns

Autonomous agents extend large language models (LLMs) by wrapping them in reasoning loops, memory storage, tool executions, and communication fabrics.

## Core Pillars of Agentic Systems

1. **Reasoning & Planning**: How agents break down goals and decide actions.
   - Read [Reasoning & Planning](planning.md).
2. **Context & Memory**: Storing short-term execution traces and long-term semantic knowledge.
   - Read [Context & Memory](memory.md).
3. **Multi-Agent Systems (MAS)**: Protocols for dividing work, routing tasks, and achieving consensus.
   - Read [Multi-Agent Orchestration](multi_agent.md).
4. **Retrieval-Augmented Generation (RAG)**: Equipping agents with custom document access.
   - Read [Retrieval-Augmented Generation](rag.md).
